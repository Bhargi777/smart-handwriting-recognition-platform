# Smart Handwriting Recognition Platform

A full-stack machine learning project for recognizing handwritten digits using CNNs with a FastAPI backend and Next.js frontend.

## Features
- Train and evaluate deep learning models on MNIST dataset
- Real-time digit recognition through drawing canvas
- Upload images for batch prediction
- API with confidence scores
- Dashboard with model metrics and performance visualizations

## Tech Stack
- **Backend**: Python, FastAPI, TensorFlow/Keras
- **Frontend**: Next.js, React, TailwindCSS
- **ML**: CNN architecture with regularization techniques
- **Deployment**: Docker, Docker Compose, Vercel
